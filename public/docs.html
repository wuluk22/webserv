API Documentation

This server provides the following capabilities:

Upload Files

Endpoint: POST /upload

Description: Allows users to upload files to the server.

Request: Multipart form data with a file input named "file".

Check Server Status

Endpoint: GET /api/status

Description: Returns the current status of the server in JSON format.

Sample Response:

{
    "status": "online",
    "uptime": "3 hours, 22 minutes",
    "requests_served": 1452
}

Static File Serving

Endpoint: GET /static/<filename>

Description: Serves static files from the server's static directory.

Example: GET /static/index.html
